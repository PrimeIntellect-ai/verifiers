### Installation
We recommend to install the Prime CLI to interact with the environments.

```bash
# install uv
curl -LsSf https://astral.sh/uv/install.sh | sh
# install the prime CLI
uv tool install prime
```

### Skills
To equip your agent with the necessary knowledge, we highly recommend to install the skills of this repository:
```bash
prime lab setup
```
The interactive installer will guide you through the necessary steps. The skills are more comprehensive than these docs, which are meant for human consumption.
