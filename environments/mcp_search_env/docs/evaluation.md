# Evaluation

Source: `docs/overview.md` and `docs/evaluation.md`.

Local evaluations run through `prime eval run`. A typical command is
`prime eval run my-env -m openai/gpt-5-nano`, which evaluates a model and saves
results locally.

Evaluation results can be inspected with `prime eval tui`, which opens a
terminal run browser for environments, models, runs, and rollout details.
