## End-User Lab Workspace Notes

Use this guidance in projects created through `prime lab setup`.

- Run `prime lab setup` before environment work begins. It is the entrypoint for Prime Lab workspaces.
- Use `prime <command> --plain` to get better formatted command outputs.
- Use `prime lab sync` to refresh skills, docs, templates, and projections.
- Use `prime lab doctor` to check workspace structure, configuration, managed skills, and agent binaries.
